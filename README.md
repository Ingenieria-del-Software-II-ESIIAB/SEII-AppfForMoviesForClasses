# SEII-AppfForMoviesForClasses Repository Overview.

## Purpose
This repository serves as a **template for a student software engineering project** focused on developing a movie-related application. It is designed to guide student teams through the collaborative development process as part of their coursework.

## Course Context
- Part of the **Software Engineering II** course.
- Offered by the **University of Castilla-La Mancha**.
- Intended for student teams to build and customize a movie app project as a class assignment.
